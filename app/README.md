# 🎮 Interactive Frontend App Launcher, Neon Tetris, Othello & Typing

本プロジェクトは、外部アセット（画像や音声ファイル）や外部フレームワーク（React/Vue等）を一切使用せず、**HTML5 Canvas**、**CSS3 3D-Transforms**、および **Web Audio API** のみを用いて構築された、極めて軽量で拡張性の高いブラウザアプリ＆ゲームコレクションです。

フロントエンドのコア技術（ゲームループ、リアルタイム描画、Web音響合成、ゲームAIアルゴリズム、文字入力判定、モバイル最適化、レスポンシブデザイン）の実力証明を目的としたポートフォリオです。

## 🛠️ 技術的特徴（ポートフォリオにおけるアピールポイント）

### 1. プログラミング用語特化型タイピング判定 ＆ リアルタイム評価 (`app/typing/`) 【NEW】
- **リアルタイムWPM ＆ 正確率（Accuracy）計測ロジック**: 
  - 打鍵ごとにリアルタイムで WPM（Words Per Minute：1分間あたりのワード打鍵速度）と正確率（ミス入力比率）を計算する状態管理ロジックをJavaScriptで実装。
  - ゲーム終了時に打鍵速度と正確性からランク（SSS, SS, S, A, B, C）を動的判定するスコアリングアルゴリズム。
- **デベロッパー向け辞書データベース**:
  - `const` などの予約語から、`aws_s3_bucket` などのインフラ用語、`observability` などのモダンSRE用語まで、IT・開発に特化した単語群と意味（解説）を自動出題。
  - 次に入力すべきキーの視覚的ハイライトおよびキーボードガイダンス機能。

### 2. ブラウザ完結型のゲームAIアルゴリズム (`app/othello/`)
- **Minimax法 ＋ Alpha-Beta（アルファベータ）枝刈り**を搭載した本格的な思考AIを実装。
- **評価関数テーブル（Heuristic Grid Weights）**: オセロの定石（角の保護、X打ち/C打ちの回避、辺の確保）を100〜-50点までの重要度テーブルで重み付けし、AIが論理的かつ戦略的に意思決定を行うロジックを実装。
- **難易度別深さ探索（Depth Search）**: Easy (1手読み), Normal (3手読み), Hard (5手読み) を探索。
- ブラウザのメインスレッドをブロックしない非同期ウェイト（ダミー長考）の実装により、プレイヤーにストレスを与えずに「AIが思考している」リアリティのあるUXを提供。

### 3. CSS 3D-Transforms によるディスクフリップ (`app/othello/`)
- 石（ディスク）が裏返る表現として、CSS 3D 空間（`perspective: 800px`, `transform-style: preserve-3d`）を定義。
- 石の表面（Player: シアン）と裏面（AI: ピンク）の要素を背中合わせで完全に隠し（`backface-visibility: hidden`）、Y軸回転（`rotateY`）のトランジションアニメーションにより、非常に滑らかでダイナミックな「石が裏返る3D演出」を実現。

### 4. HTML5 Canvas レンダリングエンジン (`app/tetris/`)
- `requestAnimationFrame` に基づくゲームループによる 60fps レンダリング。
- Canvas のシャドウ効果（`shadowColor`, `shadowBlur`）を利用した動的なブロックの発光エフェクト。
- 次に落ちてくるブロック（`Next`）、キープできるブロック（`Hold`）、および現在の落下位置を正確に示す「**ゴーストブロック（落下位置の予測投影）**」のリアルタイム計算。

### 5. Web Audio API による動的効果音合成
- 外部の `.mp3` や `.wav` ファイルを一切ロードせず、ブラウザ上で**リアルタイムに音波（ノコギリ波、三角波、サイン波）を合成**して効果音を生成する超軽量設計。
- **タイピング用効果音**: メカニカルキーボードを打っているような小気味良い打鍵音（周波数の急減衰）、ミス時の警告ブザー音（ノコギリ波の合成）、単語クリア時のクリア音を合成。
- **オセロ向け効果音**: 石を置いたときの物理的な打鍵音、連続フリップ発生時の回転質感の表現、勝敗決定時の和音（メジャーコード / マイナーコード）の合成処理。

### 6. ガラスモーフィズムUI ＆ 完全レスポンシブ
- CSSの `backdrop-filter` とグラデーションを組み合わせた、透明感のある**ガラスモーフィズム (Glassmorphism) デザイン**。
- スマートフォンのタッチスクリーンに最適化した「**ダブルタップズーム防止機能付きモバイルボタン**」の実装。
