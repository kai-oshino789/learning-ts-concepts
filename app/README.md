# 🎮 Interactive Frontend App Launcher & Neon Tetris

本プロジェクトは、外部アセット（画像や音声ファイル）や外部フレームワーク（React/Vue等）を一切使用せず、**HTML5 Canvas**、**CSS3**、および **Web Audio API** のみを用いて構築された、極めて軽量で拡張性の高いブラウザアプリ＆ゲームコレクションです。

フロントエンドのコア技術（ゲームループ、リアルタイム描画、Web音響合成、モバイル最適化、レスポンシブデザイン）の実力証明を目的としたポートフォリオです。

## 🛠️ 技術的特徴（ポートフォリオにおけるアピールポイント）

### 1. HTML5 Canvas レンダリングエンジン (`app/tetris/`)
- `requestAnimationFrame` に基づくゲームループによる 60fps レンダリング。
- ネオンスタイルを再現するため、Canvas のシャドウ効果（`shadowColor`, `shadowBlur`）を利用した動的なブロックの発光エフェクト。
- 次に落ちてくるブロック（`Next`）、キープできるブロック（`Hold`）、および現在の落下位置を正確に示す「**ゴーストブロック（落下位置の予測投影）**」のリアルタイム計算と描画。

### 2. Web Audio API による動的効果音合成
- 外部の `.mp3` や `.wav` ファイルを一切ロードせず、ブラウザ上で**リアルタイムに音波（ノコギリ波、三角波、サイン波）を合成**して効果音を生成する超軽量設計。
- 以下の操作に合わせた異なる音響をプログラム制御：
  - 移動音（三角波の周波数スイープ）
  - 回転音（サイン波の周波数増幅）
  - ラインクリア音（同時消去数に応じた周波数シフト ＋ バンドパスフィルター）
  - ゲームオーバー音（ノコギリ波の周波数減衰）

### 3. ゲームロジック ＆ 衝突判定
- 2次元配列（10x20）によるフィールドの状態管理。
- テトリミノの「回転時の壁キック処理（Wall Kick）」の簡易実装による操作感の向上。
- 消去したライン数に応じたレベルアップおよび落下速度（ミリ秒単位）の動的スケールシステム。
- テトリスの公式ルールに準拠したスコアリングシステム（1〜4ライン同時消去時のボーナス倍率）。

### 4. ガラスモーフィズムUI ＆ 完全レスポンシブ
- CSSの `backdrop-filter` とグラデーションを組み合わせた、透明感のある**ガラスモーフィズム (Glassmorphism) デザイン**。
- PCでのキーボード操作に加え、スマートフォンのタッチスクリーンに最適化した「**ダブルタップズーム防止機能付きモバイルボタン**」の実装。

## 🎮 テトリス操作方法

| 操作 | PC (キーボード) | モバイル (画面ボタン) |
| :--- | :--- | :--- |
| **移動** | `←` / `→` (左右矢印) | `←` / `→` ボタン |
| **ソフトドロップ** | `↓` (下矢印) | - |
| **ハードドロップ (即座に落とす)** | `Space` (スペースキー) | `↓` ボタン |
| **回転 (時計回り)** | `↑` (上矢印) | `↻` ボタン |
| **ホールド (キープ)** | `C` または `Shift` キー | `H` ボタン |
| **ゲーム開始 / リトライ** | 画面中央のボタンをクリック | 画面中央のボタンをタップ |
